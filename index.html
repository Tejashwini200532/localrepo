Hi! This is a small project I created to practice HTML, CSS, and GitHub usage. It demonstrates a basic repository page layout with some files and Git commands.

Files in this repository:

Commands I used to set this up:

# Clone the repo
$ git clone https://github.com/Tejashwini200532/my-git-project.git
$ cd my-git-project

# Add files and commit
$ git add .
$ git commit -m "Initial commit with basic HTML and CSS"

# Push to GitHub
$ git push origin main
